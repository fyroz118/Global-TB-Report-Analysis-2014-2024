# Global Tuberculosis (TB) Report Analysis (2014–2024)

## Project Overview
This repository contains the analysis of global TB notifications and incidence data from 2014 to 2024, focusing on age and sex distribution. The data is sourced from WHO annual TB reports.

## Files Included
- tb_age_sex_2014_2024.xlsx : Cleaned data, pivot tables, and charts.

- "Global Tuberculosis Report 2014-2024.docx"

## How to Use
- Open the Excel file to explore data tables and visualizations.
- Review the report for detailed explanations and findings.

## Data Source
World Health Organization Global Tuberculosis Reports (2014–2024): [https://www.who.int/teams/global-tuberculosis-programme/tb-reports](https://www.who.int/teams/global-tuberculosis-programme/tb-reports)
